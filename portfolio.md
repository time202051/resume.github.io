## 个人作品

### ol-base-components

**项目类型**: 企业级开发框架

**文档**: [https://time202051.github.io/baseCom.github.io/](https://time202051.github.io/baseCom.github.io/)

**技术栈**: Vue2 + Element UI

**核心功能**:
- 无需联调的 CRUD 页面自动生成
- 交互式命令行方式生成完整页面
- VSCode 插件直接生成代码

**项目亮点**:
- npx 脚本一键生成 API 接口和模块文件
- 支持多级表头、动态列显示隐藏
- 自动从 Swagger 生成表头字段
- 丰富的组件库（表格、表单、弹窗、搜索框等）
- 极大提升开发效率，几秒完成页面开发
