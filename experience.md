## 江苏正泰泰杰赛智能科技有限公司

**前端开发工程师** | 2020.05 - 2022.06

### 项目一：云组态升级成 Vue3

**项目简介**: 这是一款低代码产品，公司核心自研项目，通过拖拽实现页面的搭建（因为 Vue2 的云组态过于庞大，升级成 Vue3 提升性能，项目从 0 开始参与设计和编写）

**技术栈**: Vue3 + TypeScript + Pinia + Vue Router + Axios

**项目亮点**:
1. **Vue3 升级**: 升级 Vue3 中的不兼容的 Vue2 语法，例如插槽 slot 等
2. **组件挂载**: 组件的挂载方式用 component 动态生成组件，并根据组件位置数据通过定位在画布上渲染，组件挂载四象限实现组件的拖拽缩放旋转等
3. **状态管理**: 组件数据过于庞大，通过自定义单例 hook 实现全局状态管理去管理画布上所有组件及数据，Pinia 保存一些不那么大但全局都会使用的数据和方法
4. **主题切换**: 调研并重写切换主题。将之前 Vue2 的行内样式去除（过于繁琐可读性差），使用 HTML5 的 data-* 实现。先写两套黑白主题用到的所有数据，数据是 CSS 变量 -- 开头，再在 CSS 样式中通过 var(--xxx) 使用
5. **PSD 映射**: 任意一个 PSD，通过第三方根据解析 PSD 数据，并分析解析后的数据，根据位置属性名称属性等映射成项目中矩形（矩形中可添加图片）和文字组件
6. **数据上传**: 项目的庞大导致上传接口的数据过于庞大。通过上传 zip 压缩包方式解决。获取数据时候再解析，使用第三方的 jsZip 先将本地 json 数据转成 blob 格式并通过 jsZip 塞入 zip 中，再 append 到 formData 中，通过 formData 塞给接口，从而实现数据的上传
7. **导入导出**: 项目导入导出，通过第三方 file-saver 实现
8. **菜单图标**: 菜单自定义图标。在编辑菜单弹窗中通过选中图标实现修改菜单图标功能（包括 Element Plus 的颜色选择器实现颜色的修改）。通过 require.context() 读取 icon 文件下所有的 icon，通过 Object.keys() 获取所有 icon 的名称数组，通过循环 svg-icon 组件实现在那时所有的 icon
9. **VueUse 应用**: 项目中使用 VueUse，例如 useFullscreen、onClickOutside、useMouse、useElementSize、useMouseInElement、usePointer、useVModel、useVModels 等等
10. **数字拖拽**: 数字的按住上下拖拽实现修改的需求，通过 VueUse 的 useMouse、usePointer 和一些 watch 监听鼠标按下和放开，在按下并拖拽的时候获取起始位置差通过计算并赋值
11. **Excel 导入导出**: file-saver/xlsx/script-loader 实现导入导出到 Excel
12. **数据优化**: 优化项目通用数据，抽离到 static 文件中统一管理
13. **自定义指令**: 封装自定义指令例如 v-width、v-height、v-color、v-bgColor、v-padding、v-margin 等等指令
14. **CSS 工具类**: 通过 less 的 each 循环实现自定义 CSS 类名如 m10、b10、w100 等等
